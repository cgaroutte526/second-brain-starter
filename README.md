# Second Brain Starter

Build your own AI Second Brain — a local notes vault that Claude Code reads, writes, and remembers across sessions. Works on **Windows and Mac**.

## What you need
- A work computer (Windows or Mac)
- Claude Code (free to install — uses your company Claude Pro plan)
- ~15 minutes to start

## Start here
1. Read **[docs/what-and-why.md](docs/what-and-why.md)** — what it is and why (2-min read).
2. Follow the quickstart for your OS:
   - **[Windows](docs/quickstart-windows.md)**
   - **[Mac](docs/quickstart-mac.md)**
3. In short: install Claude Code → **Download ZIP** (green "Code" button above) → copy the `template/` folder to your Documents and rename it → open it in Claude Code → run **`/setup`**.

## What's in here
- `template/` — the blank vault you copy and make your own.
- `docs/` — the guide (what & why, quickstarts, troubleshooting).
